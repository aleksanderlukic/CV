# CV – Aleksander Lukic

Personal CV website built with plain HTML and CSS.

## Features

- **SV/EN language toggle** – switch between Swedish and English; saved in `localStorage`
- **Dark/Light mode toggle** – saved in `localStorage`
- **Download CV** – opens browser print dialog for clean PDF export
- **Contact modal** – click "Kontakta mig / Contact me" to open email, phone and LinkedIn in a modal
- **Dyslexia-friendly typography** – Inter font, 1.85× line height, optimised spacing and anti-aliasing
- **Fully responsive** – mobile, tablet and desktop
- **Clean print layout** – white background, visible ref cards, no dark boxes

## Tech Stack

Plain HTML, CSS and vanilla JavaScript. No build tools or frameworks required.

## Usage

Open `index.html` in a browser directly, or serve it with any static file server.

## Photo

Place your profile photo as `Bild1.jpg` in the project root.
